# Buddha Bowls GitHub Pages Theme

## Setup Instructions

1. Copy all files to your GitHub Pages repository
2. Update _config.yml with your site information
3. Add the front matter to your HTML pages:

```
---
layout: default
title: "Page Title"
---
``
4. Push to GitHub - theme applies automatically!

## File Structure

- _config.yml - GitHub Pages configuration
- _layouts/default.html - Main layout wrapper
- assets/css/theme.css - All styling
- Your HTML pages - Add front matter and they're themed!

## Changing the Theme

Edit assets/css/theme.css to change:
- Colors (--primary-color, --secondary-color, etc.)
- Fonts
- Spacing
- Any other styling

All pages update automatically!
